PROJECTS

Yelp Recommendation System

Xgboost Collaborative Filtering MinHash LSH Jaccard Pearson

A hybrid recommendation system was built using Item-Based Collaborative Filtering and XGBoost to predict the ratings for a pair of user and business. RMSE threshold of 0.98 was met. In addition, similar items discovery was done using Local Sensitive Hashing (LSH) and Jaccard similarity.

Market Basket Analysis for Frequent Items:

Market Basket Analysis Frequent Items Apriori Savasere, Omiecinski, Navathe (SON) algorithm

Apriori and SON algorithms were implemented to find pairs of items that are frequently bought by users in in Ta Feng grocery dataset. This was built using Python and on top of Apache Spark framework. The implementation was further extended to find frequent triples, quadruples, and so on.

Community Detection

Girvan-Newman Modularity BFS Betweenness

A social network graph was constructed using Yelp dataset in which the nodes represented the users and an edge existed if two users reviewed the same business. Girvan-Newman algorithm was implemented to calculate betweenness of edges and form communities based on modularity values.

Big Data Clustering

Bradley-Fayyad-Reina (BFR) Kmeans Discard set Compression set Retained set

Bradley-Fayyad-Reina (BFR) algorithm and K-Means clustering algorithm were used to cluster a synthetic dataset with 10 features.

Stream Mining

Bloom Filter Flajoret Martin

Bloom Filter and Flajolet-Martin algorithms were implemented to estimate whether a user id in the data stream was shown before and to estimate the number of unique users within a window in the data stream.

Multi-Label Multi-Class Classification using SVM

SVM SMOTE Hamming Loss

Multi-class and multi-label classification was done on Anuran Calls dataset using SVM algorithm with Gaussian kernels. SMOTE technique was used as a remedy for class imbalance. Hamming loss, precision, recall, ROC and AUC were used in order to evaluate the classification task.

Linear Regression Analysis of Combined Cycle Power Plant Data

Linear Regression Hypothesis Testing

Linear Regression was used to predict the net hourly electrical energy output for a Power Plant, using features such as Ambient Temperature, Ambient Pressure, Relative Humidity, and Exhaust Vacuum.

Supervised, Semi-Supervised, and Unsupervised Learning on Breast Cancer Dataset

L1-penalized SVM K-means Monte-Carlo

This dataset had two output classes: "Benign" and "Malignant" with 30 attributes. L1 penalized SVM was used to classify the data. In the next step only 50% of positive and negative classes were used to perform semisupervised learning. Finally k-means clustering was done on the whole training set to label test data.
